# Next.js Admin Dashboard

A modern admin dashboard built using Next.js, TypeScript, Material UI, Zustand, and Axios.

---

## Features

- Authentication Login
- Protected Routes
- Users Module
- User Detail Page
- Products Module
- Product Detail Page
- Search Functionality
- Pagination
- Dynamic Routing
- Zustand State Management
- Responsive UI
- Neon Dark Theme

---

## Tech Stack

- Next.js
- TypeScript
- Material UI
- Zustand
- Axios
- Tailwind CSS

---

## Installation

```bash
npm install
```

---

## Run Project

```bash
npm run dev
```
